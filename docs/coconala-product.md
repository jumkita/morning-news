# ココナラ向け商品設計（厳選ニュース）

最終更新: 2026-07-13

## 1. 立ち位置

| 層 | 役割 | ユーザー体験 |
|----|------|--------------|
| 朝刊（asahan） | **網羅確認** | 今日の見出しを漏れなく見る |
| 厳選枠（本機能） | **時短選別** | 何を先に見るべきか分かる |
| ココナラ 100円/月 | **継続サポート** | 経歴を踏まえた選球眼を毎月借りる |

課題は「情報が多いこと」ではなく、「**何を見なくてよいか分からないこと**」。

## 2. 重要度の掲載基準（実装済み）

見出し・要約テキストを語彙照合し、機械的に付与する（推測文は書かない）。

| 重要度 | ラベル | 基準のイメージ |
|--------|--------|----------------|
| A | 必読 | 金利・為替・原油・制裁・規制・半導体・EC/Amazon・自動車サプライ・市場ショックなど、意思決定に直結しやすい論点 |
| B | 確認推奨 | 人事・賃金・消費・物流・DX・エネルギーなど、押さえると時短になる論点 |
| C | 後回し可 | スポーツ・エンタメ・雑報寄り。網羅確認には残すが厳選枠からは外しやすい |

ページ構成:
1. **厳選（最大10件・A/B優先）** … 先に読む
2. **網羅（全件）** … 漏れ確認

## 3. ココナラ商品案（100円/月）

### 商品名（案）
ストラテジスト視点の朝刊厳選ニュース（月額）

### 提供内容
- 毎朝の厳選10本（重要度A/B＋「なぜ見るか」）
- 月1回の視点メモ（例: 今月の見るべきテーマ3つ）
- 公開ページ（asahan）の見方ガイド

### やらないこと
- 投資助言・個別銘柄推奨
- 有料記事全文の転載・再配信
- 「全部読め」の情報洪水

### 公開範囲・会員壁（必須開示）
- ダイジェストは各メディアの **無料見出し・リード（要約／description等）** 中心
- 有料会員向け本文は取得・再配信しない
- リンク先は日経など、会員登録・有料会員が必要なサイトを含む場合がある
- リンクを開いても全文が読めない（会員壁）ことがある。朝刊側では「見出しのみ」等で明示する
- 正確性は各メディア公式ページの原文を正とする

### 価格の意図
100円/月は「新聞代未満で、選別コストを外注する」位置づけ。  
価値の本体は情報量ではなく、**読み飛ばしの権限を渡すこと**。

## 4. 経歴との接続（訴求）

- Amazon / EC実務と自動車周辺クライアント視点で「今日の実務に効くか」を優先
- マクロ（為替・原油・金利）は価格・仕入れ・広告・在庫判断の前提として必読扱い
- サッカー速報などは網羅側に残し、厳選から外す
